.htaccess
#########################
simplecontent nutzt eine .htaccess. Das heißt, wir machen aus einem Link wie 
simplecontent-cms.org/?post=any-post&id=2 einen SEO-friendly-Link wie: simplecontent-cms.org/manual. 
Internetseiten, die solche URLs haben, werden höher bei Google, Bing und co. gelistet.
Wenn dein Webserver keine .htaccess-Dateien unterstützt, benenne die Datei in htaccess.txt um oder
lösche sie. Info: Unter XAMPP funktioniert es bestens!

TinyMCE 
########################
Wir nutzen TinyMCE (ältere Version), einen tollen Editor der fast so viel Funktions-
umfang wie Word hat.

CHMOD und Uploads
########################
Bei manchen Webhostern ist es so, das du Ordner "berechtigen" musst, damit simplecontent Dateien 
uploaden kann. Wenn es bei dir der Fall ist, wähle den Befehl chmod777 für den Ordner 
simple-content/upload.
